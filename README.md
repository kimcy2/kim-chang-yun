## 프로젝트 개요
1. REVIA (AI 패션 인텔리전스 플랫폼)
- **개요:** AI가 사용자의 옷 이미지를 분석해 코디, 스타일, 키워드를 추천  
- **기술:** React · Spring Boot · Python FastAPI · OpenAI GPT-4o · K-Fashion AI 모델  
- **특징:** AI 화보 자동 생성 / Glass Tone UX / 쇼핑몰 연동  
- **성과:** 이미지 분석 → AI 스타일 추천 → 시각적 패션 매거진 자동 생성  
- **소감:** 디자인 완성도와 AI 연동을 동시에 완성하며, “보여줄 수 있는 완성도”를 목표로 개발 :contentReference[oaicite:3]{index=3}

---

2 LOCALR7 (7번 국도 맛집 리뷰 커뮤니티)
- 개요: 동해안 여행지 중심의 맛집 리뷰 & AI 챗봇 통합 플랫폼  
- 기술: React · Spring Boot · OracleDB · Google Maps API · Gemini AI  
- 주요 기능: 구글/카카오 소셜 로그인, AI R7봇, 지역별 맛집 정보, 관리자·리뷰 시스템  
- 성과: Google Gemini API로 AI 맛집 추천 및 리뷰 요약 기능 구현  
- 소감: 팀장으로서 기획, 디자인, 조율 전담 — 협업과 일정 관리의 중요성 체득 :contentReference[oaicite:4]{index=4}

---

3 NEXTROZI (기업 일정·거래 관리 플랫폼)
- **개요:** 거래처, 재고, 발주 관리 등 B2B 업무를 한 화면에 통합  
- **기술:** React · Spring Boot · MySQL · 협업도구 Notion / Discord  
- **특징:** ERP 스타일의 데이터 관리 / 직원 조직도 및 발주 시스템  
- **성과:** 직관적 UI + 데이터 흐름 통합으로 업무 효율성 향상  
- **소감:** 협력과 역할 분담의 중요성을 배우며 팀 리더십 강화 :contentReference[oaicite:5]{index=5}

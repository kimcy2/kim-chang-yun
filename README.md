## 🔹 1. REVIA | AI 패션 스타일 추천 플랫폼

[![REVIA 시연 영상](https://img.shields.io/badge/🎥_시연영상-Click_to_Watch-blue?style=for-the-badge)](./REVIA.mp4)

> AI가 옷 이미지를 분석하고, 스타일·키워드·코디를 자동 추천하는 패션 AI 플랫폼

**📌 프로젝트 개요**
- 사용자가 업로드한 옷 이미지를 분석해  
  색상, 패턴, 스타일, 키워드를 추출하고 코디 문장을 생성  
- AI 결과를 “패션 매거진 스타일”로 시각화

**🧰 주요 기술**
- React · Spring Boot · FastAPI  
- OpenAI GPT-4o · K-Fashion AI 모델  
- Zustand · Axios · Styled-components  

**💡 특징**
- Glass-Tone UX 디자인  
- AI 화보 자동 생성  
- 패션 키워드 기반 추천 쇼핑 연동  

**🏆 성과**
- AI 이미지 분석 → 텍스트 요약 → 추천 코디 자동화  
- “디자인 완성도 + 기술 구현력”을 모두 보여준 대표작

## 🔹 2. LOCALR7 | 지역 맛집 리뷰 & AI 챗봇 커뮤니티

[![LOCALR7 시연 영상](https://img.shields.io/badge/🎥_시연영상-Click_to_Watch-green?style=for-the-badge)](./LOCALR7.mp4)

> 동해안 7번 국도를 중심으로 한 지역 맛집 커뮤니티 & AI 챗봇 서비스

**📌 프로젝트 개요**
- 사용자 리뷰 데이터를 기반으로  
  AI가 맛집을 추천하고 대화로 안내하는 플랫폼  
- 지도 기반 맛집 탐색과 커뮤니티 리뷰 기능 통합

**🧰 주요 기술**
- React · Spring Boot · OracleDB  
- Google Maps API · Gemini AI  
- 소셜 로그인(Google, Kakao)

**💡 특징**
- AI 챗봇 “R7봇”으로 맞춤형 맛집 대화 추천  
- 관리자 페이지 + 사용자 리뷰 관리 시스템  
- 실시간 리뷰 요약 및 AI 응답 기능  

**🏆 성과**
- Gemini API로 리뷰 자동 요약 기능 구현  
- 협업 및 일정 관리 리더십 향상

## 🔹 3. NEXTROZI | 기업 일정·거래·재고 관리 플랫폼

[![NEXTROZI 시연 영상](https://img.shields.io/badge/🎥_시연영상-Click_to_Watch-orange?style=for-the-badge)](./NEXTROZI.mp4)

> 기업 내부의 거래처, 재고, 발주, 일정 관리를 한 화면에서 처리하는 B2B 웹 시스템

**📌 프로젝트 개요**
- 복잡한 유통 데이터를 통합 관리  
- 거래처/발주/직원 조직도 기능으로 ERP형 서비스 제공

**🧰 주요 기술**
- React · Spring Boot · MySQL  
- Notion · Discord 협업  

**💡 특징**
- 거래처 등록 / 발주 계획 / 일정 관리  
- ERP 스타일 UI 설계  
- 실시간 데이터 반영  

**🏆 성과**
- UI/UX 개선과 데이터 흐름 통합으로 업무 효율성 향상  
- 팀 리더십 및 협력 기반 개발 프로세스 완성
